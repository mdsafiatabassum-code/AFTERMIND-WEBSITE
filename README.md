# AfterMind

## Project Description

AfterMind is an AI-powered emotional storytelling and journaling platform designed to help users express emotions and experience self-reflection through immersive cinematic interactions.

## Features

* AI emotional journaling
* Cinematic storytelling
* Mood tracking
* Personalized reflections

## Tech Stack

* React
* Firebase
* Tailwind CSS
* AI API

## Installation

Clone repository:
git clone (https://github.com/mdsafiatabassum-code/AFTERMIND-NEW-)

Install dependencies:
npm install

Run project:
npm run dev
